About Fire Alarm System

The Fire Alarm System is an IoT-based solution designed to detect fire, smoke, and environmental changes in real time. The system employs a combination of temperature, humidity, and smoke sensors to monitor conditions in a designated area. In the event of smoke or fire detection, the system triggers an alarm to notify users instantly. Our goal is to provide a reliable, real-time alerting system that can be integrated into residential, commercial, and industrial spaces to enhance safety and prevent fire-related accidents.

Technology Used

This project utilizes a combination of hardware and software components, including:

  • Hardware: ESP32 microcontroller, MQ-2 smoke and gas sensor, DHT11 temperature and humidity sensor.
  • Software: The backend is built with Django, and the frontend is developed using HTML, CSS, and JavaScript. Data is stored in Firebase Realtime Database for easy access and updates.
  • Communication: The ESP32 communicates with Firebase to send sensor data and receives control commands from the web interface.

How It Works

The Fire Alarm System continuously monitors the temperature, humidity, and smoke levels in a monitored environment. If any of the parameters exceed predefined thresholds, an alarm is triggered. The system provides real-time updates via a web dashboard, showing sensor readings and alarm status. Users can manually override the alarm or turn it off as needed. The system can be accessed remotely, offering flexibility and convenience.

Key Features

  • Real-time monitoring of temperature, humidity, and smoke levels.
  • Automatic fire detection and alert system.
  • Manual override and alarm control.
  • Data logging and storage in Firebase for historical analysis.
  • Responsive web interface for ease of use.

Our vision is to create a safer environment by offering an affordable and easy-to-implement fire detection solution.
